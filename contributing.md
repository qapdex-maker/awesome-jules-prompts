# Contributing 🤝

Thank you for considering contributing to **Awesome Jules Prompts**!

## How to Contribute

- <span aria-hidden="true">📋</span> We have a <a href="pull_request_template.md" aria-label="Read our Pull Request Template">Pull Request Template</a> to help you get started.
- <span aria-hidden="true">📝</span> Add a new prompt to the appropriate section in <a href="README.md" aria-label="Read our Main README">README.md</a>.
- <span aria-hidden="true">🎯</span> Keep prompts concise and clear.
- <span aria-hidden="true">🪄</span> Submit a <a href="https://github.com/qapdex-maker/awesome-jules-prompts/pulls" target="_blank" rel="noopener noreferrer" aria-label="Open a Pull Request (opens in a new tab)">pull request</a> and describe your change.

## Style Guide

- <span aria-hidden="true">💻</span> Use inline code formatting for prompt examples (`// like this`).
- <span aria-hidden="true">🗂️</span> Group related prompts under the right heading.
- <span aria-hidden="true">🔝</span> Include an accessible `Back to top` link at the end of any new or
  significantly expanded sections in
  <a href="README.md" aria-label="Read our Main README">README.md</a> to
  maintain easy navigation
  (e.g., `<a href="#table-of-contents" aria-label="Back to Table of Contents">Back to top ↑</a>`).
- <span aria-hidden="true">💡</span> Feel free to suggest new categories if needed.

Let’s build this together!
