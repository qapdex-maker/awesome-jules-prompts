# Security Policy

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not** open a public issue.
2. Contact the maintainers privately (e.g., via the contact information provided in the repository profile).
3. Provide a detailed description of the vulnerability and steps to reproduce it.

We will acknowledge your report and work to address the issue as quickly as possible. Thank you for helping keep this project secure!
